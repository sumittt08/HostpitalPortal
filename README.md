# Hospital Management Portal

## Overview
A Full-Stack ASP.NET Core MVC application developed for the HCL TechBee Assessment.
It manages patient registrations and provides a dashboard for doctors to view and search patient records.

## Tech Stack
* **Backend:** C# (.NET 8.0)
* **Frontend:** HTML5, Bootstrap, Razor Views
* **Database:** SQL Server LocalDB with Entity Framework Core

## Features
* **Patient Registration:** Create new patient records (Name, Age, Symptoms).
* **Doctor Dashboard:** View all patients in a grid.
* **Search:** Filter patients by name in real-time.
